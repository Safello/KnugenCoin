What is KnugenCoin?
----------------

Litecoin is a royal version of Litecoin using scrypt as a proof-of-work algorithm.

134 miljoner KnC kommer skapas, ett mynt för varje krona i H.M. Carl XVI Gustafs förmögenhet. En blockhalvering tar 0.335 år (representerar en 200-del av H.M. Carl XVI Gustafs ålder). Varje block kommer innehålla 3900 mynt och skapas en gång i minuten. Fint som snus!
